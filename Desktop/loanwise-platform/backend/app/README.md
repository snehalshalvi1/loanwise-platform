# LoanWise – Loan Risk Analysis Platform

A full-stack machine learning web application that predicts loan risk and provides user-specific analytics dashboards.

## 🚀 Features
- User authentication (signup/login)
- ML-based loan risk prediction
- User-specific dashboards (no data mixing)
- Risk classification (Low / Medium / High)
- Confidence scoring
- CSV export functionality
- Search + filtering

## 🛠 Tech Stack
Frontend: React, Tailwind CSS, Recharts  
Backend: FastAPI, SQLAlchemy  
Database: SQLite  
ML: Python (custom model)

## 📊 Highlights
- Built end-to-end ML pipeline from input → prediction → visualization
- Solved real-world issue: user-specific data isolation
- Designed clean, modern dashboard UI
- Implemented analytics KPIs for business insights
